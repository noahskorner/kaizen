# Kaizen

## Prerequisites

- Volta
  - https://docs.volta.sh/guide/getting-started
- Docker Desktop
  - https://docs.docker.com/desktop/install

## Getting Started

1. Clone the repository

   ```
   gh repo clone noahskorner/kaizen
   ```

2. Install dependencies

   ```
   npm install
   ```

3. Run the application

   ```
   npm run dev
   ```
